OXI ONE MKII Mastery Tutorial

A structured, hands-on curriculum for turning the OXI ONE MKII from a complex sequencer into a precise composition and performance instrument. Every lesson contains explicit button moves, concrete step patterns, and repeatable drills.

How to use this tutorial

This is not a reference manual. It is a sequence of deliberate exercises. Do the modules in order once, then return to individual modules as drills. The goal is operational mastery: you should be able to create, mutate, perform, arrange, save, and reconnect a project without hunting through menus.

Ground rule: the OXI ONE MKII does not produce audio by itself. Connect it to a synth, drum machine, modular voice, iPad app, or DAW instrument before judging the result.

Notation

[Button] means press a front-panel button. (Knob 1) through (Knob 4) mean the four encoders from left to right. [Shift] + [Button] means hold Shift while pressing the other control. A “step” means a column position in the pattern. A “row” means pitch, track, probability, or function depending on the current mode.

1. Orientation, project safety, and setup

Sequencers

The instrument contains 8 independent sequencers. Treat each as a musician: bass, lead, pad, kick, snare, hats, modulation, or generative texture.

Patterns

A project can contain patterns for the sequencers. A pattern can run up to 128 steps, viewed through the page buttons [16], [32], [48], and [64]; double-taps access later pages.

Project discipline

Project = total state: sequencers, patterns, songs, settings, mappings. Save often using [Shift] + [Save].

First power-on drill

  1. Insert the microSD card before powering on. This avoids missing folder structure problems later.
  2. Connect power or use battery. Connect MIDI USB, TRS MIDI, Bluetooth MIDI, or CV/Gate to one sound source.
  3. Press the side power button. After boot, double tap [Back] to guarantee you are at the home sequencer view.
  4. Press [Load]. Use (Knob 1) to inspect project slots. Do not overwrite a meaningful slot yet.
  5. Press [Save]. Move to an empty or sacrificial slot with (Knob 1), press (Knob 1), choose save, press (Knob 1) again.
  6. Rename the project later, but for now quick-save with [Shift] + [Save] after every completed module.
Mastery standard: you can load, save, quick-save, back out of menus, and return home without thinking. This matters more than creativity at the beginning.

2. Sequencer grammar: steps, pages, length, scale, and root

The OXI grid is context-sensitive. In Mono/Poly/Chord, columns are time and rows are pitch or chord degrees. In Multitrack, rows are tracks. In Stochastic, columns are note slots rather than ordinary step positions. Mastery begins by knowing what the grid means before placing data.

Global navigation drill

  1. Tap [1] to select sequencer 1. Its sequencer button should indicate focus.
  2. Hold [Shift] + [1] to open sequencer settings. On setup page 1, set Mode to MONO. Set output and MIDI channel or CV/Gate mapping according to your rig.
  3. Double tap [Back] to return home.
  4. Hold [Shift] and turn (BPM / Knob 1) to set tempo to 120.
  5. Hold [Shift] and turn (Scal / Knob 3) to select Minor. Hold [Shift] and turn (Root / Knob 4) to select C2.
  6. Use (Oct / Knob 2) to place C2/F2/G2 within visible rows. Press [Keyboard] only when you want a playable keyboard layout; press it again or back out to return to sequencer editing.

3. Mono mode: build a precise melodic phrase

Mono mode is the fastest route to understanding the instrument. One track, one note per step, 128 possible steps. The grid behaves like a piano-roll constrained by the chosen scale.

Exercise 3A — six-step minor cell

  1. Select [1]. Set mode to MONO with [Shift] + [1], then return home.
  2. Set 120 BPM, C minor, root C2.
  3. Set pattern length to 6: hold [End] and tap the pad in column 6. Start should remain step 1.
  4. Enter notes: column 1 = C2, column 3 = F2, column 5 = G2. Tap one pad in each specified column at the row matching that pitch.
  5. Press [Play]. You should hear a looping 6-step phrase that cycles against 4/4 in a slightly offset way.

Exercise 3B — accumulator motion

  1. Hold the step-3 pad, then tap [Page] until you reach the step submenu containing accumulator controls.
  2. Still holding step 3, turn (Amt / Knob 1) to +1st.
  3. Hold the step-5 pad and set accumulator amount to -1st.
  4. Listen for gradual small pitch displacement. Then reset either accumulator by holding the relevant step and setting amount back to 0.

Exercise 3C — copy the phrase and make counterpoint

  1. Hold [Copy] and tap [1].
  2. Hold [Paste] and tap [2]. Sequencer 2 now carries the phrase.
  3. Select [2]. Set its output channel to a second sound.
  4. On sequencer 2, set length to 8 steps. Move the G2 note from step 5 to step 7. Lower velocity on step 3 to 60. This creates separation from sequencer 1.

4. Pattern shaping: velocity, gate, repeats, probability, swing, and time division

A beginner places notes. An advanced user shapes event behavior. Work through the same six-step phrase and deliberately alter one parameter at a time.

Exercise 4A — make a phrase breathe

  1. On sequencer 1, hold step 1 and set velocity to 110, gate to 80%.
  2. Hold step 3 and set velocity to 72, gate to 32%.
  3. Hold step 5 and set velocity to 95, gate to 55%.
  4. Set global swing by holding [Shift] and turning (SWNG / Knob 2) to 54–58%. Do not overdo it yet.
  5. Press [Play] and compare against straight 50% swing.

Exercise 4B — controlled repeats

Exercise 4C — time division contrast

  1. Copy sequencer 1 to sequencer 3.
  2. Select sequencer 3 and set it to a pluck or percussive sound.
  3. Use [Shift] + [Page] to increase time-division resolution, or [Shift] + [LFO] to reduce it. Also test [Shift] + [Init] and [Shift] + [End] for doubling/halving behavior.
  4. Set sequencer 3 to a faster division than sequencer 1. Mute/unmute to hear how identical notes become a new layer through timing alone.

5. Poly and Chord modes: harmony without guesswork

Poly mode

Poly mode is free-form polyphony: multiple notes can exist on the same step. Use it when you want exact voicings, clusters, two-note intervals, or written parts.

Chord mode

Chord mode sequences chord structures, inversions, and voicings. Use it when you want fast harmonic progressions and scale-based chord thinking.

Exercise 5A — Poly: exact voicing loop

  1. Select sequencer 4. Hold [Shift] + [4] and set mode to POLY. Set output to a polyphonic synth.
  2. Set scale to C minor, root C2, length 16.
  3. Enter this four-chord loop. For each step, tap multiple rows in the same column to form the voicing.

Exercise 5B — Chord: stored chord palette

  1. Select sequencer 5. Set mode to CHRD. Use a chord-capable synth patch.
  2. Set scale/root to C minor. Press [Keyboard] to access chord selection.
  3. Use columns 13–16 to choose chord type, voicing/inversion, spread, and chord bank. Audition chords.
  4. Store four chords by holding a keyboard chord pad and then holding an empty top-row memory pad 1–12 until it lights.
  5. Return to sequencer view. Hold [Keyboard], tap stored chord 1, release, then tap step 1. Repeat chord 2 at step 5, chord 3 at step 9, chord 4 at step 13.
  6. On step 13, add x2 repeat and lower probability to 80% for an occasional cadence variation.
Harmony drill: for 10 minutes, build a progression in Poly mode, then rebuild it in Chord mode. The difference trains exact note entry versus harmonic abstraction.

6. Multitrack mode: drum programming and multi-lane composition

Multitrack mode gives eight independent monophonic tracks. The default grid view maps rows bottom-to-top as tracks 1–8. Each track can have its own length, start/end, time division, and groove behavior, which makes it ideal for drums.

Exercise 6A — 16-step drum kit

  1. Select sequencer 6. Hold [Shift] + [6] and set mode to MULTI.
  2. Assign the output to a drum machine or DAW drum rack. Map tracks bottom to top: 1 Kick, 2 Snare, 3 Closed Hat, 4 Open Hat, 5 Clap, 6 Perc, 7 Tom, 8 FX.
  3. Set length to 16 steps. Enter the pattern below.

Exercise 6B — make it human without making it sloppy

  1. Hold the snare steps and set velocity around 105 on step 5 and 112 on step 13.
  2. Set closed-hat velocities alternating 70 and 55.
  3. Add probability: percussion track steps 3, 7, 11, 14 at 70–85%.
  4. Add a short repeat to the tom on step 13, but set probability to 50%. This creates a fill that appears sometimes, not always.
  5. Hold [Mute]. Use column 1 to mute rows/tracks and column 16 to solo rows/tracks while judging the groove.

Exercise 6C — polymetric hats

  1. Select track 3 closed hat. Set its end to step 15 instead of 16.
  2. Keep kick/snare at 16 steps.
  3. Let it run for at least 64 steps. The hat phase will rotate against the beat. This is the simplest route to movement without adding new notes.

7. Generators: Euclidean, random, stochastic, and controlled surprise

Generators are useful only when you set boundaries. Do not ask randomness to compose. Use generators to propose material, then edit, delete, and commit.

Exercise 7A — Euclidean percussion

  1. On the multitrack drum sequencer, select the percussion row/track.
  2. Tap [Gen] to open generator options.
  3. Create a 5-hit pattern across 16 steps. If available, rotate the pattern until it avoids the main snare on steps 5 and 13.
  4. Set velocity lower than kick/snare and probability around 75%.
  5. Commit only if it improves the groove. Otherwise clear that track and retry.

Exercise 7B — Random generator with guardrails

  1. Choose an empty Mono or Poly sequencer, not a finished one. Random generation can overwrite previous data.
  2. Hold [Shift] + [Randomizer] to open the random generator.
  3. Constrain scale/root first. Use C minor, 16 steps, low density, moderate pitch range.
  4. Generate. Immediately remove half the notes manually. Keep the best motif, not the whole output.
  5. Save the pattern only after editing it into a usable musical phrase.

Exercise 7C — Stochastic sketch

  1. Select an empty sequencer and set mode to STOC.
  2. Understand the grid differently: columns represent note candidates rather than ordinary timeline steps.
  3. Enter four notes: C2, Eb2, G2, Bb2. Assign probabilities: C = 100%, Eb = 70%, G = 85%, Bb = 40%.
  4. Use random pitch/rhythm columns cautiously. Listen until a phrase emerges.
  5. When it produces a phrase worth preserving, copy or recreate the useful result in Mono/Poly/Multitrack for deterministic editing.
Rule of controlled surprise: randomization should produce raw material, not final form. The final form comes from deletion, probability design, and performance decisions.

8. Modulation mastery: lanes, LFOs, Quick Assign, and musical automation

Each sequencer has modulation lanes for parameter movement over time, plus LFOs for cyclic movement. A modulation lane is like a step sequencer for values rather than notes. It can drive external MIDI messages, CV-style changes, and internal sequencer parameters when mapped.

Exercise 8A — MIDI CC filter sweep lane

  1. Select the melodic sequencer that controls a synth with a filter cutoff CC.
  2. Tap [Mod] to open modulation lanes.
  3. Tap (Knob 1) to select modulation lane 1. If needed, configure its destination to the synth filter cutoff CC.
  4. Set lane length to 16 with [Init]/[End] as needed.
  5. Draw this value curve into lane 1: steps 1–4 = 20%, 5–8 = 45%, 9–12 = 80%, 13–16 = 35%.
  6. Play the pattern. If the synth jumps too abruptly, smooth the lane or reduce modulation amount at the destination.
Lane 1 values: 20 20 20 20 | 45 45 45 45 | 80 80 80 80 | 35 35 35 35

Exercise 8B — internal probability modulation

  1. Pick a percussion or melodic fill step with probability below 100%.
  2. Use the modulation matrix or Quick Assign to map a modulation source to trigger probability or randomization where available.
  3. Set modulation amount small first: ±10–20%. Avoid maximum depth until you understand the effect.
  4. Listen for structural variation over 2–4 bars, not just one loop.

Exercise 8C — LFO on a destination

  1. Press [LFO] to open LFO settings for the selected sequencer.
  2. Select LFO 1. Set wave to triangle, rate to 1/2 or 1 bar, amount low.
  3. Set destination to a filter, timbre, pan, or internal parameter. If a parameter shows as modulatable, use [Shift] + long press the parameter knob for Quick Assign.
  4. Increase amount until you clearly hear motion, then reduce it by one third. This prevents novelty from overwhelming the pattern.

9. Performance tools: Keyboard, Arp, Flow, Roller, Perform, Snapshots

Exercise 9A — Keyboard recording

  1. Select a Mono or Poly sequencer.
  2. Press [Keyboard]. Use the grid as a playable note surface.
  3. Press [Rec] + [Play] for step recording, or arm recording with [Rec] + [Stop] if you want recording to begin on note input.
  4. Record four notes, stop, return to sequencer view, and edit gate/velocity per step.

Exercise 9B — Flow as alternate reality

Flow steps only trigger when Flow is held. Use them for fills, alternate notes, or temporary rhythmic injections.

  1. On sequencer 1, hold [Flow] and tap step 4 at G#2.
  2. Hold [Flow] and tap step 6 at A#2.
  3. Play the sequence without holding Flow. The base phrase should remain unchanged.
  4. Hold [Flow] during playback. The added notes should appear only during the held interval.

Exercise 9C — Perform grid transposition

  1. Press [Perf]. Each row represents a sequencer performance control area.
  2. For Mono/Chord/Poly sequencers, use pads around the central neutral column to transpose. Move sequencer 1 up, sequencer 2 down, then return both to neutral.
  3. Hold [Shift] while pressing a performance pad where rotation is available. Listen for rhythmic offset without changing pitch.
  4. Practice 16 bars: bars 1–4 neutral, bars 5–8 transpose sequencer 2, bars 9–12 use Flow, bars 13–16 return and mute one layer.

Exercise 9D — Snapshot discipline

  1. Create a stable “home” state: all important sequencers unmuted, transpositions neutral, modulation reasonable.
  2. Capture a snapshot.
  3. Create a performance chaos state: mutes, transpositions, Flow, random perform, roller, and modulation changes.
  4. Recall the snapshot. The audience hears risk; you retain a safety net.

10. Arranger: turn patterns into a song

The Arranger is where loops become form. A song consists of organized pattern launches and slots. Use it once your patterns are strong enough to survive repetition.

Exercise 10A — create a five-part arrangement

  1. Create or duplicate pattern variations for bass, drums, chords, and lead. Keep names simple: A, A2, B, Break, End.
  2. Press [Arranger] to enter arranger view. Use [Shift] + [Arranger] to enable/disable arranger operation as needed.
  3. Create these sections: Intro 8 bars, Groove A 16 bars, Variation 16 bars, Break 8 bars, Final 16 bars.
  4. Assign patterns to slots. For Intro, use drums without kick, pad only, no lead. For Groove A, add kick/bass/chords. For Variation, add lead and modulation. For Break, remove kick and use Flow fills. For Final, restore full pattern and add performance transposition.
  5. Set launch quantization so pattern changes occur musically, normally on bar boundaries for structured songs.
  6. Save project. Then play the arrangement end-to-end without touching anything. Mark weak transitions and fix pattern content rather than over-editing the arranger.

11. Connectivity: MIDI, CV/Gate, clock, and troubleshooting logic

MIDI setup checklist

  1. Choose transport and note path: USB MIDI, TRS MIDI, Bluetooth MIDI, or a splitter/host setup.
  2. For each sequencer, hold [Shift] + [Seq], set output and MIDI channel.
  3. Match the receiving synth/DAW track to that channel. If there is no sound, first check channel mismatch before changing notes.
  4. Use the MIDI input monitor when diagnosing external controllers: hold [Shift] + hold [Back] for more than 2 seconds.

CV/Gate setup checklist

  1. Patch CV output to oscillator pitch input and Gate output to envelope/gate input.
  2. Hold [Shift] + [Gen] to access CV/Gate mapping.
  3. Set voltage behavior appropriate to the target. Gate outputs are configurable, so do not assume every module wants the same level.
  4. Use one simple C2 step first. If pitch is wrong, solve calibration/scaling before adding complexity.

Mastery checklist

Navigation

You can select any sequencer, change mode, set channel/output, return home, and save without hesitation.

Editing

You can place, tie, clear, copy, paste, shift, length-limit, and page through steps accurately.

Expression

You use velocity, gate, repeats, probability, swing, groove, and time division as intentional composition tools.

Modes

You know when to use Mono, Poly, Multitrack, Chord, Stochastic, Matriceal, Domino, Saga, and MIDI Player.

Modulation

You can build a modulation lane, assign a MIDI/internal destination, use an LFO, and diagnose missing modulation.

Performance

You can perform with mute, Flow, transposition, rotation, Roller, random perform, and snapshots without losing the song.

Arrangement

You can transform patterns into a multi-section song with meaningful transitions.

Connectivity

You can solve no-sound, wrong-channel, wrong-clock, and wrong-CV/Gate problems methodically.

Final exam: In 45 minutes, create a new project from zero with one bass sequence, one drum multitrack, one chord sequence, one modulation lane, one Flow variation, one performance snapshot, and a five-section arrangement. Save it internally and copy it to the microSD card. If you can do that without referring to the manual, you are operationally fluent.
